# Search Console / AI Discoverability Notes

## Submit after deploy

1. Open Google Search Console.
2. Select `https://spacegleam.co.jp/`.
3. Go to `Sitemaps`.
4. Submit:

```text
https://spacegleam.co.jp/sitemap.xml
```

5. Use URL Inspection and request indexing for:

```text
https://spacegleam.co.jp/
https://spacegleam.co.jp/blog/
https://spacegleam.co.jp/blog/why-space-gleam-builds-own-services/
```

## Files added for crawlability

- `robots.txt`
- `sitemap.xml`
- `llms.txt`
- Blog pages with Article JSON-LD
- Organization JSON-LD on the top page
