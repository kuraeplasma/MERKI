DIFFsense

契約レビューSaaS

契約書の差分確認や修正案作成を支援するAI契約レビューSaaS。

開発の背景

契約書レビューは専門知識が必要で、弁護士や法務担当者の工数が大きく、レスポンスも遅くなりがちでした。AIで差分確認と修正案作成を自動化し、スピードと精度を両立するサービスとして開発しました。

主な機能

契約書差分比較AI修正案の提示条文の要約リスク検出電子署名・履歴管理

技術スタック

  • Next.js / TypeScript
  • OpenAI API
  • PostgreSQL / Supabase
  • AWS (S3 / Lambda / CloudFront)
  • Stripe / Clerk（認証・決済）

開発期間

初期モデル構築：約2週間

継続的に改善・機能追加を実施

現在の状況

  • 法務事務所・企業の法務部で利用中
  • 継続的に機能追加・改善を実施
  • より高度なAIレビュー機能を開発中
